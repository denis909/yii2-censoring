Модуль для автоматической замены нецензурных слов по словарю.  

За основу был взят модуль из форума FluxBB который был полностью переписан на Yii 2 и доработан.

Из оригинального кода используется только пара регулярных выражений и структура БД.

GPL version 2 or higher
